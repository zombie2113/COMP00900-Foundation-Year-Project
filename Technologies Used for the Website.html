Hypertext Markup Language (HTML)

Computer Science 13
Group 13
































Definition: HTML stands for Hypertext Markup Language, a foundational coding language used for building web pages.

Basic Elements in HTML :
  1. < p >- -paragraph
  2. < h >- -heading
  3. < br / >- -start new line
  4. < title >- -define document title
  5. < hr / >- -horizontal line
  6. < strong >- -display bold font
  7. < i >- -italic font style
  8. < q >- -inline quotation
  9. < li >- -list
          

 Although additional programming languages and techniques are required to achieve results beyond  static web pages (features like animated graphics, interactive forms, and photo slideshows), but no  matter what HTML is a key, foundational step in web development.
